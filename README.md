# OpenClaw Shared

Public, reusable OpenClaw documentation, templates, and examples for peers building their own OpenClaw setups.

## Contents

- `docs/directives_safeguards_shareable.md` - generalized safeguards reference for sharing and adaptation
- `docs/directives_safeguards_template.md` - ready-to-customize template with search-and-replace placeholders

## Publishing Model

These files are exported from a private workspace using explicit public-share markers, then reviewed before commit and push to this public repository.

That is less convenient than dumping everything into a repo, but vastly less stupid.
